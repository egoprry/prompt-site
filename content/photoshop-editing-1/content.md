# Photoshop

Quick Photoshop pass to remove AI-render look

## Lens Blur
**Filter > Blur > Lens Blur**

- **Depth Map Source:** None
- **Iris Shape:** Hexagon (6)
- **Radius:** 4–8 (start at 6)
- **Blade Curvature:** 20–30
- **Rotation:** 0
- **Specular Highlights:** Brightness 2–5, Threshold ~230
- **Noise:** 0
- Use **More Accurate** for the final render
- **After:** add a layer mask, paint blur off the figure + near floor

## Grain
**Filter > Camera Raw Filter > Effects > Grain**

- **Amount:** 15–25
- **Size:** 8
- **Roughness:** 50
- Apply to a merged copy on top so grain covers the whole image

> **Quick alternative:** Filter > Noise > Add Noise — 3–5%, Gaussian, Monochromatic ✓
